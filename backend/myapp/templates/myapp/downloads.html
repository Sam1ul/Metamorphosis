How to Use vuln_bank_demo.zip on Linux or Windows with XAMPP

Steps for Linux:

  1. Download the ZIP file using the button above.
  2. Open your terminal and navigate to the download location:
    cd /path/to/protected_files
  3. Extract the ZIP file:
    unzip vuln_bank_demo.zip
  4. Start XAMPP:
    sudo /opt/lampp/lampp start
  5. Copy the extracted folder into XAMPP's htdocs:
    cp -r vuln_bank_demo /opt/lampp/htdocs/
  6. Open the project in your browser:
    http://localhost/vuln_bank_demo

Steps for Windows:

  1. Download the ZIP file using the button above.
  2. Locate the ZIP in your Downloads folder and extract it (Right‑click → “Extract All…”).
  3. Open XAMPP Control Panel and start:
    • Apache
    • MySQL (if needed by the labs)
  4. Move the extracted folder into your XAMPP htdocs directory:
    C:\xampp\htdocs\
  5. Your folder should now look like:
    C:\xampp\htdocs\vuln_bank_demo\
  6. Open your browser and go to:
    http://localhost/vuln_bank_demo

⚠ Important: Only use vulnerable training applications on machines you own or control. Never deploy them on public networks.
